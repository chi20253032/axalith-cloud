# Axalith Cloud

PenguinMODで公開される予定のクラウドシステムです。

## 概要
Axalith Cloudは、PenguinModおよびHTMLベースのアプリケーションに高度なクラウド機能とセキュリティを提供するインフラストラクチャです。2,000人規模のユーザーを支えるバックエンド基盤として設計されています。

## 主な機能
- **Firebase 連携**: リアルタイムデータベース（Firestore）を活用した高速なデータ同期。
- **高度なUI (Liquid Glass)**: HTML/CSSによる、透過・ぼかしを駆使した美麗なデザインシステム。
- **二段階認証 (2FA)**: セキュアなユーザー認証プロセス。
- **ハイブリッド設計**: 
  - **PenguinMod**: 基本UIおよびゲームロジックを担当。
  - **HTML/JS**: 高度なUI、セキュリティ、データベース通信を担当。

## 開発ロードマップ
1. **Axalith Cloud Core**: 基盤および認証システムの構築（現在進行中）
2. **BT2 連携**: Scratchプロジェクトにおけるクラウド変数活用
3. **BT3 統合**: 次世代プロジェクト「BT3」への全面的な組み込み（半年後予定）

## 開発環境
- **Hardware**: Alienware 16 Aurora
- **Editor**: Cursor / Anti-Gravity
- **Framework**: Firebase SDK, Tailwind CSS (Material Design 3)
